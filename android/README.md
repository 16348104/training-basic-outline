# Android

### &sect; 1. 课程教学目标

通过本课程的学习，使学员全面掌握 Android 应用程序开发的基础知识；能够编写简单规范的 Android 原生应用程序。

本课程还应使学生掌握 Android 基本开发工具，初步掌握 Android 程序调试技术。
